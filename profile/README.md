# Sprint Soluções em Tecnologia

**Consultoria especializada em BI, Analytics e Engenharia de Dados**

## O que fazemos

Transformamos dados em decisões através de soluções modernas e escaláveis para empresas que precisam ir além de dashboards estáticos.

### Áreas de atuação

- **Visualização e modelagem semântica** — Power BI, Microsoft Fabric, DAX, Power Query
- **Cloud e Data Warehousing** — Azure (Data Factory, Synapse), Google Cloud (BigQuery), Snowflake, Databricks
- **Engenharia de Dados** — dbt Core/Cloud, pipelines de ingestão, modelagem dimensional, ELT/ETL
- **Integração e automação** — APIs REST, integração com ERPs (Omie, TOTVS, SAP), Python

### Tecnologias

`Power BI` · `DAX` · `dbt` · `BigQuery` · `Snowflake` · `Python` · `SQL` · `Microsoft Fabric` · `Next.js`
